SNEAK Evolution

A modernised reconstruction of the Snarkles.Net Encryption Assortment Kit (v1.28). Originally authored by Snarkles, this tool was a staple of early 2000s security research but eventually became redundant due to the collapse of MD5/DES security standards. This update preserves the original logic while integrating modern AES-256-CBC encryption and external rainbow table lookup capabilities via remote API.
